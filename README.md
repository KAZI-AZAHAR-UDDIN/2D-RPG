## Project Setup
1. Clone the repository:
git clone git clone  https://github.com/KAZI-AZAHAR-UDDIN/2D-RPG.git

2. Navigate to the project directory:
cd 2D_RPG

3. Install dependencies:
npm install

5. Run the development server:
npm run dev

6. Build and start the app:
npm run build
npm start
